Memory Game

O jogo da memória é uma aplicação interativa desenvolvida em JavaScript, onde os jogadores devem encontrar pares de cartas idênticas. As cartas começam viradas para baixo, e ao clicar, elas se revelam. O desafio é memorizar as posições e encontrar todos os pares no menor número de tentativas. O design responsivo garante uma jogabilidade suave em diferentes dispositivos, tornando o jogo divertido e estimulante para todas as idades
